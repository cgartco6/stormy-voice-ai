# ⚡ Stormy Empire OS

### 🛑 CRITICAL SETUP
1. **Add to Home Screen:** Open your GitHub URL in Chrome (Android). Tap the **3-dots** and select **"Add to Home Screen."**
2. **First Tap:** You must tap the microphone button once when you open the app to give Stormy "permission" to talk.
3. **Assets:** Your photos MUST be in the `assets/` folder and named exactly `bg-1.jpg` (Normal) and `bg-2.jpg` (Furious).

### ✅ KEY COMMANDS:
- **"Play [Song] by [Artist]"**: Searches Spotify.
- **"Call [Number]"**: Dials the phone.
- **"Navigate to [Place]"**: Opens Google Maps.
- **"Bok Radio"**: Plays 98.9 FM.
