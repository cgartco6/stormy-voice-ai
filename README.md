# ⚡ Stormy AI | Cape Town Rebel

A sarcastic, mobile-first Voice AI assistant designed to run as a Web App on Android.

## 🚀 Deployment Instructions

1. **Upload Files**: Ensure `index.html` and the `assets/` folder are in the root of your repository.
2. **The .nojekyll File**: Make sure there is an empty file named `.nojekyll` in the root. This ensures GitHub Pages serves your `assets/` folder correctly.
3. **Photo Check**: Your images **MUST** be named exactly `bg-1.jpg` and `bg-2.jpg`. If they are `.png` or `.JPG` (uppercase), rename them or update the code.
4. **Enable Pages**: Go to **Settings > Pages** in your GitHub repo and set the branch to `main`.

## 📱 How to make her "REAL" (Mandatory)

To allow Stormy to open your **Music**, **Notes**, and **Phone Dialer**, you cannot just use the browser tab.
1. Open your GitHub Pages link in **Chrome on Android**.
2. Tap the **three dots (⋮)** in the top right.
3. Select **"Add to Home Screen"**.
4. Close Chrome and open **Stormy** from your phone's home screen icon.

## 🛠️ Troubleshooting Music
If the music player doesn't open:
* Ensure you have a default music app (Spotify, YouTube Music, or Samsung Music) installed.
* When you first try the command, watch for a system popup asking for permission to open another app. Select **"Always"**.
